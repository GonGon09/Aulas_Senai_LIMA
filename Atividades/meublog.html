Meu Blog de HTML: Coisas que Aprendi Até Agora

1. Estrutura básica de uma página HTML

Aprendi que toda página HTML precisa de uma estrutura mínima para funcionar corretamente. Ela começa com uma declaração que indica ao navegador que tipo de documento ele vai ler. Depois, existem partes que guardam informações sobre a página, como o título, e outras que contêm todo o conteúdo visível da página.

2. Títulos e parágrafos

Descobri que podemos organizar o conteúdo usando títulos e parágrafos. Existem níveis diferentes de títulos, do mais importante ao menos importante. Os parágrafos servem para separar o texto em blocos, tornando a leitura mais fácil.

3. Links e imagens

Aprendi a adicionar links e imagens às minhas páginas. Para criar um link, precisamos informar o endereço do site que queremos acessar. Para imagens, é necessário indicar o caminho do arquivo e escrever uma pequena descrição, caso a imagem não carregue.

4. Listas

Listas são ótimas para organizar informações. Podemos criar listas ordenadas, que numeram os itens, ou listas não ordenadas, que usam marcadores. Cada item dentro da lista representa uma informação separada.

5. Formatação de texto

Também aprendi maneiras de destacar o texto. Por exemplo, podemos deixar o texto em negrito para dar mais importância, ou em itálico para dar ênfase. Isso ajuda a chamar atenção para as partes mais importantes do conteúdo.
